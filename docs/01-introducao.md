# Introdução ao Go

Go é uma linguagem de programação de código aberto criada pela Google, focada em simplicidade, eficiência e concorrência nativa.

:::info
**Por que usar Go?**
* Compilação direta para código de máquina.
* Estaticamente tipada, mas com sintaxe limpa.
* Gerenciamento de memória automático.
:::

## Instalação e Versão
Conforme o roadmap oficial, para verificar a instalação do Go no seu ambiente, utilize o comando:

```bash
go version
